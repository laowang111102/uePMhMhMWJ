# Python人事管理系统 python926

## 项目概述

本项目是基于Python语言开发的人事管理系统，采用Django作为后端框架，vue作为前端框架，以及MySQL作为数据库支持。系统旨在为用户提供全面、高效的人事管理功能。

## 技术栈

- 编程语言：Python
- 开发工具：PyCharm
- 后端框架：Django
- 前端框架：vue
- 数据库版本：MySQL 5.7+
- 数据库工具：Navicat、SQLyog

## 功能模块

- 首页
- 用户管理
- 员工管理
- 公司信息管理
- 招聘信息管理
- 应聘申请管理
- 反馈结果管理
- 培训信息管理
- 参加培训管理
- 培训记录管理
- 考勤信息管理
- 请假信息管理
- 加班信息管理
- 弹窗提醒管理
- 系统管理

## 系统角色

- 系统管理员
- 员工
- 招聘负责人
- 培训负责人

## 运行环境

- Python环境
- Django框架支持
- vue.js
- MySQL数据库

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 安装vue.js及相关依赖
4. 配置MySQL数据库
5. 运行后端服务
6. 运行前端服务
